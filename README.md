# Calculating Nth Fibonacci number

## Usage

To calculate Nth Fibonacci number type: `./fibo N`

## Switches

```txt
--all,     -A   Prints sequence up to 93
--help,    -H   Prints help
--version, -V   Prints version number
```

## Build

```bash
git clone https://github.com/yousefvand/fibonacci.git
cd fibonacci
make
```
